# 高级 A+(Premium / ebc-premium)模块规格

> 实跑后台抓取(2026-06-26,美国站)。单篇内容最多 **7** 个模块。
> `*` = 必填;图片尺寸为后台标注的目标像素(宽:高);文案为字符上限。
> 「点击添加图片」= 图片上传触发点。所有模块均带「AI 效果润色」。

## 纯图横幅适用(素材是整幅带文字的图,优先用这些)

| 模块 | 图片(px) | 文案字段 | 备注 |
|---|---|---|---|
| **高级完整图片** | 图片* 1464:600 | 标题≤80、正文≤300 | 单张整幅图,最常用 |
| 高级、简单的图像轮播 | 图片* 1464:600 ×(2–6 面板) | 每面板:正文≤200、按钮文本≤12、ASIN≤10 | 多图轮播,可带按钮跳转 |
| 高级规则轮播 | 图片* 1464:600 ×(2–5 面板) | 标题≤100、每面板 导航文本*、文本≤20 | 带导航标签的轮播 |
| 高级导航轮播 | 图片* 1464:600 ×(2–5 面板) | 每面板:导航文本*≤25、标题≤50、正文≤100、副标题 | |
| 高级热点1 | 图片* 1464:600 | (在图上打热点标注) | 纯图+热点 |
| 高级热点2 | 图片* 1464:600 | 标题*≤80、正文≤300 | 图+热点+文字 |

## ⚠️ 素材适配提醒

测试素材是 **1594×986(≈1.62:1)/ 1672×941(≈1.78:1)** 的整幅图。
后台整幅图模块要 **1464×600(2.44:1)**,比测试图**更宽更扁** → 直接上传会触发**裁切**(上下被裁掉一部分)。
这属于"选型/适配"范畴(Claude Code/运营决定:接受裁切、还是改用 650:350 的双图、或重新出图)。程序只负责按选定模块上传。
